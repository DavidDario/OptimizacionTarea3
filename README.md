## Instrucciones de implementación 

**Requisitos**
1. Poseer una version de python superior a python3.12. 
2. Contar con una cuenta de github.
3. Tener algún editor de texto (Block de Notas, Vim, Neovim, Visual Studio Code,  etc).

--- 
**Configuración de entorno virtual**

1. Revisar que se cuenta con la libreria de virtual enviroment. Si no es el caso:
> pip install virtualenv
2. Crear un nuevo directorio y entrar en este. 

3. Dentro del directorio utilizar el siguiente comando :
```bash
  python -m venv .env 
```

4. Para activar el ambiente virtual este dependera de tu sustem 
  - Windows:
    - .env\Scripts\activate
  - MacOS/Linux:
    - source .env/bin/activate

5. Una vez activado el ambiente virtual descargar las librerias debes utilizar el siguiente comando:

```bash
  pip install -r requirements.txt 
```
